广西重点矿业企业“走出去”座谈调研会在南宁召开

2022年3月17日11:16 | 来源：中国-东盟矿业信息服务平台

广西重点矿业企业

“走出去”座谈调研会在南宁召开

应自治区商务厅的要求，2022年3月16日下午，中国—东盟地学合作中心（南宁）召集广西海外矿业产业联合会部分会员单位代表，在南宁召开广西重点矿业企业“走出去”座谈调研会。自治区商务厅党组成员周密、自治区地矿局副局长黎四龙出席了会议。
座谈会上，与会企业代表围绕本企业生产经营情况、资源保障程度、企业对海外资源的需求和“走出去”的困难、问题与诉求等方面畅所欲言，在海外矿业法律政策、资源核查与合作、矿业活动风险、融资授信、政策性与商业性保险、出境手续办理、国际贸易与矿产品物流回运等方面，希望得到政府相关部门的支持和帮助。座谈过程中，商务厅调研组对企业反映的问题与诉求在政策和市场层面解决的渠道、路径作了解答。
周密对中国—东盟地学合作中心（南宁）组织召开的这次座谈会给予了充分肯定，认为座谈会对今后相关政策的起草有着重要的意义。他指出，国家一直鼓励企业“走出去”开拓国际市场、开发利用国外资源。矿业经济是广西经济重要的组成部分，但资源供给保障不足是当前工矿企业普遍面临的困难，因此要从全球的视觉去审视和寻求问题的解决，“走出去”合作开展海外资源勘查和开发。各企业海外业务不一样，“走出去”遇到的问题、面临的挑战也不同，但所反映的困难或诉求，相当部分是因为信息不对称、没有形成合力等原因形成。包括商务部门在内，政府层面一直致力于支持、帮助企业“走出去”，如广西建立了“桂企出海”服务平台、安排有相关扶持资金、海外政策性保险、海外投资前置报备与风险提示等，政府层面今后还将进一步研究出台相关政策，加大对企业“走出去”的扶持力度。因此，希望矿业企业与政府相关部门加强联动、互通信息，用好政策。同时也希望企业联合金融部门、行业协会、上下游全产业链形成合力，以解决海外矿业活动中遇到的诸如资金、物流、产业链业务协同等方面的问题。
黎四龙对本次座谈取得的成效也予以充分肯定。他希望各企业对会上所反映的具有普遍性、共性的问题进一步梳理，向上反映，多渠道寻求相关问题的解决。同时希望今后邀请国际法律研究、国际物流、商保、金融等机构和企业参与研讨，增强针对性和服务企业“走出去”的能力。

创新发展

自治区人大财经委国资监督处、商务厅对外经济合作处、东盟国家及区域合作处、中信保广西分公司业务三处、广西社科院数量经济研究所和自治区地矿局对外合作处负责人参加了会议，广西海外矿业产业联合会秘书长单位中国—东盟地学合作中心（南宁）、会长单位南方锰业集团、副会长单位中铝广西分公司、中金广西公司、吉利百矿集团、百色百金集团、柳工集团、大晋国际矿业等共11家企业负责人及其相关部门负责参加了座谈。
